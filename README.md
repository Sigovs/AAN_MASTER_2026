# AAN Master 2026 — Design System + Homepage Prototype

A token-first design system and homepage prototype for **AAN Motors** (All Auto
Network), a luxury/exotic auto dealer. Everything — color, spacing, radius, type,
shadow, motion — flows from one token file, so a dealer "clone" re-skins the whole
site by editing tokens, not components.

> **Prototype, not production.** This is a design/UX prototype for review. It is
> not the shipping WordPress theme; the real production theme lives separately
> (see `reference/`, which is client-owned and never published). Demo photos are
> placeholders.

## Run it

```bash
npm install          # dev deps: sass, live-server, playwright
npm run watch        # compile src/scss → dist/css/main.css (live)
npm run serve        # live-server on http://localhost:5173  (second terminal)
# one-off build:
npm run build
```

Open **`index.html`** (the homepage) and **`design-system.html`** (the living
style guide). `dist/css/main.css` is committed so the pages render on GitHub /
Pages without a build step.

## Three skins

Toggle with the bottom-right **Skin** switcher (JS sets `data-skin` on `<html>`):

- **Obsidian** — dark, chromatically neutral (the default master).
- **Gallery** — light, neutral.
- **Midnight** — a *clone example*: the graphite base pushed cooler and deeper
  into a subtle violet-blue mood, showing how a dealer re-skins the neutral master.

The master ships neutral on purpose — premium comes from type and spacing, not a
brand hue; a clone sets `$color-primary`.

## Layout

```
src/scss/          SOURCES (SCSS only)
  aan-theme-kit.scss   the single source of truth (tokens + skin maps)
  base/ components/ sections/ pages/   partials, imported via main.scss
assets/            static: img/demo (car-1..3), js/ui.js (behavior), fonts/
dist/css/main.css  GENERATED by the sass build (committed for viewing)
reference/         READ-ONLY client material — never modify / publish
index.html · design-system.html      the two prototype pages
scripts/screenshot.mjs                Playwright self-verification
CLAUDE.md          permanent project rules (tokens, rhythm, DS-doc contract, …)
```

Bootstrap 4 provides grid/utilities/JS only; all visuals route through
`bootstrap-bridge.scss`. Placeholder fonts: Bodoni Moda (display), Archivo
(body), IBM Plex Mono (mono), Oswald (labels) — swapped in one place when the
real pair is chosen.
